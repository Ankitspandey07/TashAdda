iPhone / iPad — sideload required

Apple does not allow one-tap install like Android. Use the steps below (free Apple ID).

Download iOS IPA (v1.0.1)
  1. On a Mac/PC install Sideloadly.
  2. Connect iPhone with USB.
  3. Drag the IPA into Sideloadly → enter your Apple ID.
  4. On iPhone: Settings → General → VPN & Device Management → trust.

Or use AltStore to auto-refresh before the 7-day cert expires. Full iOS guide

Free iOS installs expire after ~7 days. Reinstall or use AltStore refresh. For the easiest experience, Android friends should use the APK button above.
